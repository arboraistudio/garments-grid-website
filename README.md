# Garments Grid

> Premium Laundry-as-a-Service (LaaS) Platform

Garments Grid is a premium on-demand platform designed for corporate and government professionals in major North American hubs. By separating logistics (Couriers) from processing (vetted home-based Launderers), our three-sided **Tri-Sync Engine** solves the $10B headache of garment care with infinite asset-light scalability.

---

## 🚀 Key Features

*   **Interactive Visual Cataloging**: A fully clickable mobile mockup simulator directly on the landing page. Users can add garments, toggle between custom Care Plans (**Full Care** vs. **Press & Fold**), and trigger camera shutter snapshots to auto-populate the manifest.
*   **Predictable Recurring Revenue (MRR)**:
    *   **Grid Plus (Executive Care)**: $14.99/mo premium membership unlocking priority delivery and turnaround discounts.
    *   **Family Subscription (Household)**: $59–$189/mo flat-rate weekly plans for busy households.
    *   **Grid Scale (Pay-by-Weight)**: $2.00/lb flexible plan for everyday wash-and-fold needs.
*   **Tri-Sync Live Telemetry**: A dashboard console displaying active order matching simulation logs.
*   **Launch Hubs**: Scaled and localized for Toronto, Montreal, Calgary, Winnipeg, Washington D.C., and New York.

## 🛠️ Technology Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router, React 19)
*   **Styling**: Vanilla CSS (Global Variables, Bento layout tokens, keyframe animations)
*   **Fonts**: Next.js Google Font integration (`Outfit`, `DM Serif Display`, `JetBrains Mono`)

## 💻 Getting Started

### 1. Installation
Clone the repository and install the dependencies:
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---

## 📦 Deployment

This project builds as a static application and is optimized for automated deployment via GitHub Pages. On every push to the `main` branch, the workflow compiles the project, runs checks, and deploys the production bundle.
